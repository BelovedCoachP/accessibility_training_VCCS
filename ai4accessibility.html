1. Alt Text Generation

Recommended Tools

  • ChatGPT: chat.openai.com (Free/Paid)
  • Google Gemini: gemini.google.com (Free)
  • Microsoft Copilot: copilot.microsoft.com (Free/Paid)

Copy-Paste Prompts

"Write concise alt text for this image for accessibility purposes."
"Describe this chart/diagram for someone who cannot see it."
"Create alt text for this [photo/graph/illustration] focusing on [key information]."
Pro Tips:
  • Always review AI suggestions for accuracy
  • Verify technical terms and names
  • Keep alt text concise (1-2 sentences)
  • Mark decorative images as decorative (don't use AI for these)

2. Video Captions & Transcripts

Recommended Tools

  • YouTube Auto-Captions: Free, built into YouTube
  • Google Meet Transcription: Free, built into Google Meet
  • Otter.ai: otter.ai (Free tier: 300 min/month)
  • Descript: descript.com (Paid)

Workflow

  1. Upload video to platform
  2. Let AI generate initial captions
  3. Edit for accuracy (budget 5-10 min per video)
  4. Export and attach to video
⚠️ Common Errors to Fix:
  • Names of people and places
  • Technical terms and jargon
  • Discipline-specific vocabulary
  • Homophones (there/their, two/too/to)
  • Punctuation and sentence breaks
Pro Tip: Create a custom vocabulary list in your captioning tool for recurring technical terms.

3. Simplifying Complex Language

Recommended Tools

  • ChatGPT or Google Gemini: For rewriting content (Free/Paid)
  • Grammarly: grammarly.com (Free/Paid - includes readability scores)
  • Hemingway Editor: hemingwayapp.com (Free web version)

Copy-Paste Prompts

"Rewrite this text at a 10th-grade reading level while maintaining accuracy."
"Break this paragraph into shorter, clearer sentences."
"Simplify this academic language for general readers."
"Explain this concept in plain language without losing meaning."

Why This Matters

  • Helps students with cognitive disabilities
  • Supports ESL learners
  • Improves comprehension for everyone
  • Reduces cognitive load

4. Document Accessibility Review

Recommended Tools

  • ChatGPT or Google Gemini: For guidance and explanations
  • Grackle for Google Docs: grackledocs.com (Free/Paid)

Copy-Paste Prompts

"What accessibility issues might this document have?"
"Give me a checklist for making this [document type] accessible."
"How do I fix color contrast issues in my slides?"
"What's the best way to structure headings for accessibility?"
"How can I make this table more accessible for screen readers?"
"What does 'logical reading order' mean and how do I check it?"
How to Use:
  • Ask AI for general guidance and explanations
  • Use Grackle or built-in checkers for actual scanning
  • Verify AI suggestions with real accessibility tools

5. Accessible PDF Creation

Recommended Tool: Grackle for Google Docs

grackledocs.com

Installation

  1. Open any Google Doc
  2. Go to Extensions → Add-ons → Get add-ons
  3. Search "Grackle Docs"
  4. Click Install

How to Use

  1. Create document in Google Docs with proper headings and alt text
  2. Go to Extensions → Grackle Docs → Launch
  3. Review flagged accessibility issues
  4. Fix issues using Grackle's guided suggestions
  5. Export as accessible PDF

What Grackle AI Checks

  • Missing alt text
  • Heading structure
  • Color contrast
  • Reading order
  • Table headers
  • Form fields

AI Prompt Library - Copy & Use These

For Images

"Write alt text for this course schedule graphic"
"Describe this lab diagram for a biology student using a screen reader"
"Create alt text for this chart showing [specific data]"
"What should I include in alt text for this campus map?"

For Videos

"Review these auto-generated captions and flag any errors"
"Create a transcript from this video lecture including speaker names and timestamps"
"What's missing from these captions for accessibility?"

For Documents

"Is this heading structure accessible: [paste your headings]"
"How can I make this table accessible: [describe table]"
"What's wrong with this link text: 'click here'"
"Suggest better link text for: [paste URL]"

For Content Simplification

"Simplify this course policy for students: [paste text]"
"Rewrite this assignment description at a 10th-grade level: [paste text]"
"Make this syllabus section clearer: [paste text]"

For Color & Contrast

"What color combinations meet WCAG 2.1 AA standards for text?"
"Is [color] on [background color] accessible?"
Note: Always verify color contrast with actual contrast checkers like WebAIM Contrast Checker

AI Best Practices Checklist

✓ DO

  • Use AI as your starting point, not your final answer
  • Always review and edit AI-generated content
  • Verify accuracy of names, terms, and technical content
  • Test results with real accessibility tools
  • Save effective prompts for reuse
  • Share successful strategies with colleagues
  • Keep learning as AI tools evolve

✗ DON'T

  • Trust AI alt text without reviewing it
  • Publish auto-generated captions without editing
  • Rely solely on AI for compliance
  • Skip verification with accessibility checkers
  • Assume AI understands your specific context
  • Share AI content without human review
  • Forget that you're the expert on your content

Troubleshooting Common Issues

Problem Solution
AI alt text is too generic Add specifics to your prompt: "Write alt text for this enrollment chart focusing on the 25% increase from 2023 to 2024"
AI captions have many errors Use the platform's custom vocabulary feature to teach it your discipline-specific terms
AI simplified text too much Adjust your prompt: "Simplify while maintaining technical accuracy" or "Rewrite at college level, not middle school"
Don't know which AI tool to use Start with free options (ChatGPT, Gemini, YouTube) before investing in paid tools
AI doesn't understand my content Provide more context in your prompt: "This is a chemistry diagram showing..."
